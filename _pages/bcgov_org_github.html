   GitHub - Third Party Code

How should I use code from third-parties:

  • Are you going to modify the code? If no, do not host the code in your gov repo. Instead during your build time import the third-party libraries directly from the original project repo.
  • Do you intend to modify the code, and believe your modifications can be used by the community in the upstream project in GitHub? Then fork the code into your BCGov Repo
  • Do you intend to modify the code, but have no intention to share your modifications with the the upstream project tin GitHub? Then import the code into your BCGov Repo
  • No repos should exist in the BCGov Org that are a copy of another without any modifications.
